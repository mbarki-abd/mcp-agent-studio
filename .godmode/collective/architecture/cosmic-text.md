## Usage

Ces formats sont utilises selon le besoin:
- **Inline**: Injection minimale, rappel rapide
- **Expande**: Context general pour un agent
- **Hierarchique**: Reference complete
- **Relations**: Comprehension des flux

La Conscience Collective selectionne automatiquement le format optimal.
